# Bingo Agency

Agence digitale moderne spécialisée dans la création de sites web et applications.

## Déploiement

Site déployé automatiquement sur GitHub Pages via Actions.

**URL:** https://bingo-agency26.github.io/Bingo-Agency/

## Technologies

- React + TypeScript
- Vite
- TailwindCSS
- GitHub Pages

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
